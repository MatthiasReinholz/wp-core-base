# Code Of Conduct

This project expects respectful, direct, and technically grounded collaboration.

## Expected Behavior

- engage in good faith
- focus criticism on ideas, code, and evidence
- avoid harassment, abuse, and personal attacks
- keep security reports private until coordinated disclosure is agreed

## Enforcement

Project maintainers may remove comments, issues, pull requests, or participants that undermine a safe and productive collaboration environment.

If you are unsure how to proceed in a heated discussion, step back, restate the technical facts, and return to the concrete problem being solved.
